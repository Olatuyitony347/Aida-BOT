# Aida-BOT
 clone repo
 
 ```
git clone https://github.com/Olatuyitony347/Aida-BOT.git
```

```
cd Aida-BOT
```

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
source ~/.bashrc
```

```
nvm install 18
nvm use 18
npm install
node -v
```

```
npm install ethers
```

```
npm install axios
```

Edit and save your ref code:
```
nano main.js
```

```
apt install screen
```

```
screen -S ai
```

```
chmod +x bot.sh
```
To Run continuously by reseting wallet files:
```
./bot.sh
```
To run once:
```
node main.js
```
